# Gym-Membership-Tracker

This is a very simple Java console project using JDBC to perform basic operations on a database.

##  Features
- Add member (INSERT)
- List members (SELECT)

##  Technologies Used
- Java 
- JDBC 
- MySQL Database

##  Files in this Repository
- `GymJDBC.java` → Java source code using JDBC
- `create_table.sql` → SQL script to create database & table
- `README.md` → Project description
